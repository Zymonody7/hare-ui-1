## 组件 API

### Attributes 属性

| 参数   | 说明         | 类型    | 可选值                     | 默认值        |
| ------ | ------------ | ------- | -------------------------- | ------------- |
| type   | 主题类型     | string  | success/warning/info/error | success       |
| center | 文字是否居中 | boolean | -                          | false         |
| title  | 提示标题     | string  | -                          | This is title |
